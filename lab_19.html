Welcome to HTML Basics

HTML (HyperText Markup Language) is the foundation of web pages. It structures content for the web.

Where is HTML Used?

  • Websites
  • Emails
  • Web Applications
  • Embedded Content

Basic Structure of an HTML Document

An HTML document consists of the following main elements:

  1. <!DOCTYPE html> - Defines the document type and version.
  2. <html> - The root element that contains all HTML content.
  3. <head> - Metadata, including the page title and linked resources.
  4. <body> - The visible content of the webpage.

Example HTML Code

        <!DOCTYPE html>
        <html>
        <head>
            <title>My First HTML Page</title>
        </head>
        <body>
            <h1>Hello, World!</h1>
            <p>This is a basic HTML page.</p>
        </body>
        </html>
